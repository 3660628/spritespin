How to use

In order to make SpriteSpin work you have to create a container where the animation should be displayed. Here i create a div container and give it the spritespin id. 

<div id="spritespin"/>

To attach the spritespin plugin to the container find it using jquery, call .spritespin and pass some options to it. 

<script type="text/javascript">
  $("#spritespin").spritespin({
    width  : 100,
    height : 100,
    "image : images/spritespin.jpg",
    frames : 36,
  });
</script>
In this example we use a spritesheet with 36 frames (images) where each hast the width and height of 100 pixels. So the image at "images/spritespin/jpg" has a width of 36 * 100 = 3600 pixels.
