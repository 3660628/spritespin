Demo 4

Here spritespin is used to create a simple image viewer. Pictures are taken from freedigitalphotos.net 

<div id="demo-4"></div>
<script">
$(function(){
  $("#demo-4").spritespin({
    width  : 400,
    height : 264,
    frames : 6,
    animate: false,
    image  : [
     "sprites/demo-4/0.jpg",
     "sprites/demo-4/1.jpg",
     "sprites/demo-4/2.jpg",
     "sprites/demo-4/3.jpg",
     "sprites/demo-4/4.jpg",
     "sprites/demo-4/5.jpg",
    ],
    behavior : "none"
  }).bind("click", function(){
    $(this).spritespin("update");
  });
});
</div>
