What is it?

Spritespin is a jQuery plugin that enables sprite animation in your website.

It requires a series of images stiched together into a single spritesheet and is able to play these images frame by frame which results in an animation. The aim of this plugin is to provide a 360 degree view of some kind of product. There is no flash needed, everything is done with javascript and the jQuery framework.

Since the Chrome Browser has problems animating large images, such as large spritesheets, the plugin can also be used with an array of single images instead.

Dependencies

The only requirement is the jQuery core framework. SpriteSpin is tested with jQuery-1.4.2.

Optional requirement is the jQuery-ui framework in order you want to control the animation with a slider.

Features

  • Animation using spritesheets
  • Interaction with mouse
  • Custom interaction
  • Support for touchpad devices
  • Easy to integrate
  • No flash needed
