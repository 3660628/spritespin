Options

Option Description Default
width The width of a single frame. 640
height The height of a single frame. 480
frames Total number of frames. 36
frame Initial frame number. 0
animate If true, starts the animation after initialize. false
loop If true, always repeats the animation.
speed Time in miliseconds between updates. 32 (8 Frames per second)
reverse If true, plays the animation backward. false
image Path to the sprite image. "images/spritespin.jpg"
onFrameChanged Event that is called when the framenumber has changed. undefined
slider Instance of a jQuery-ui Slider. undefined
interactive If true, enables mouse interaction. This allows to "drag" the animation. true
