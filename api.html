Options

Option Default Description
Dimensions
width undefined Window width (or frame width)
height undefined Window height (or frame height)
offsetX 0 Offset in X direction from the left image border to the first frames left border
offsetY 0 Offset in Y direction from the top image border to the first frames top border
frameStepX undefined Distance in X direction to the next frame if it differs from window width
frameStepY undefined Distance in Y direction to the next frame if it differs from window height
resolutionX undefined The spritesheet resolution in X direction
resolutionY undefined The spritesheet resolution in Y direction
framesX undefined Number of frames in a single row (if you user spritesheets with multiple rows)
frames 36 Total number of frames
frame 0 Initial frame number
Animation
animate false Run animation on startup
loop false Repeat animation in a loop
loopFrame 0 Indicates the loop frame. Animation will stop here after one iteration when the loop option is false
frameTime 36 Time between frame updates
reverse false If true animation is played backward
Interaction
slider undefined jQuery-ui slider instance
behavior "drag" Behavior implementation (mouseinteraction) to use.
sense 1 Sensitivity value. Used by behavior implementation e.g. for touch sensitivity. Must not be zero. Negative values are allowed (inverts the drag direction).
touchable undefined Tells spritespin that it is running on a touchable device
Appearance
image "images/spritespin.jpg" Stiched source image
preloadHtml " " Html to appear when images are preloaded
preloadBackground undefined Image to appear when images are preloaded
Events
onFrame undefined Occurs on every frame update
onLoad undefined Occurs after all images are loaded and spritespin has been initialized
panorama undefined Tells spritespin that it is showing a panorama image

Methods

Animate
Signature
.spritespin("animate", [play:boolean], [loop:boolean])
Examples
Get a value indicating whether the animation is running or not. 
.spritespin("animate")
Start the animation if not running. 
.spritespin("animate", true)
Stop the animation if already running. 
.spritespin("animate", false)
Toggle the animation. 
.spritespin("animate", "toggle")
Start the animation if not already running and keeps it looping. 
.spritespin("animate", true, true)
Start the animation if not already running and plays it only once. 
.spritespin("animate", true, false)
Loop
Signature
.spritespin("loop", [do_loop:boolean])
Examples
Get a value indicating whether the animation is looped or not. 
.spritespin("loop")
Set the loop option to true. If animation option is also true, the animation will be started. 
.spritespin("loop", true)
Set the loop option to false. Current animation will be stopped at the end of loop. 
.spritespin("loop", false)
Frame
Signature
.spritespin("frame", [value:number])
Examples
Get the current frame number 
.spritespin("frame")
Update the current frame number to 10. 
.spritespin("frame", 10)
