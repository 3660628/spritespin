Options

Option Default Description
Dimensions
offsetX 0 X offset where the frame image starts
offsetY 0 Y offset where the frame image starts
width undefined Window width (or frame width)
height undefined Window height (or frame height)
frameStep undefined Width of a single frame or step to the next frame
frames 36 Total number of frames
frame 0 Initial frame number
Animation
animate false Run animation on startup
loop false Repeat animation in a loop
loopFrame 0 Indicates the loop frame. Animation will stop here when loop is false
speed 36 Time between updates
reverse false If true animation is played backward
Interaction
slider undefined jQuery-ui slider instance
behavior "interactive" Enables mouse interaction
Appearance
image "images/spritespin.jpg" Stiched source image
preloadText "Loading" Text to appear when images are preloaded
preloadBackground undefined Image to appear when images are preloaded
Events
onFrameChange undefined Occurs on every update
onLoad undefined Occurs after images are loaded and spritespin has initialized

Methods

Animate 
.spritespin("animate")
Gets a value indicationg whether the animation is running or not. 
.spritespin("animate", true)
Starts the animation if not already running. 
.spritespin("animate", stop)
Stops the animation if not already running. 
.spritespin("animate", "toggle")
Toggles the animation 
.spritespin("animate", true, true)
Starts the animation if not already running and keeps it looping. 
.spritespin("animate", true, false)
Starts the animation if not already running and plays it only once.
Loop 
.spritespin("loop")
Gets a value indication whether the animation is looped. 
.spritespin("loop", true)
Sets the loop setting to true. If animation setting is also true, the animation will be started. 
.spritespin("loop", false)
Sets the loop setting to false. Current animation will be stopped at the end.
Frame 
.spritespin("frame")
Gets the current framenumber 
.spritespin("frame", )
Updates the current framenumber to the specified value.
